# 서비스명
내 대출 가능액 빠른 판정기

# 서비스 한 줄 설명
사용자의 소득, 현금, 무주택 여부, 생애최초 여부, 혼인 상태, 직업 형태, 나이, 주택 가격을 바탕으로
정부대출과 일반 주담대의 예상 가능 금액을 빠르게 보여주는 실거주 의사결정 웹앱

# 1단계 목표
사용자가 최소한의 정보만 입력하면 아래를 바로 보여준다.
- 예상 최종 대출 가능액
- 디딤돌 가능성 및 예상액
- 보금자리론 가능성 및 예상액
- 일반 주담대 예상 가능액
- 체증식 가능 여부
- 보유 현금을 합친 총 구매 가능 금액

# 왜 이 서비스를 먼저 만드나
대부분의 사용자는 평수, 방향, 역거리보다 먼저 아래를 궁금해한다.
- 나는 대출이 나오나
- 정부대출 대상인가
- 생애최초 혜택이 있나
- 직장인과 자영업자가 차이가 있나
- 체증식이 되나
- 그래서 결국 얼마까지 가능한가

따라서 1단계 MVP는 주택 추천보다 대출 가능액 판정을 우선한다.

# 대상 사용자
- 생애최초 주택 구입 예정자
- 신혼부부
- 실거주 목적의 아파트 매수 희망자
- 예산보다 먼저 대출 가능 금액을 알고 싶은 사용자
- 부동산 상담 전에 본인 상황을 미리 점검하고 싶은 사용자

- 본인 연소득
- 배우자 연소득
- 보유 현금
- 기존 월 부채 상환액
- 무주택 여부
- 생애최초 여부
- 혼인 상태
- 직업 형태
- 나이
- 희망 주택 가격
- 지역 구분
- 체증식 관심 여부

## 입력 항목 설명
- 혼인 상태: 미혼 / 신혼 또는 결혼예정 / 기혼
- 직업 형태: 직장인 / 자영업자 / 프리랜서 또는 기타
- 지역 구분: 수도권 또는 규제지역 / 비수도권 또는 비규제지역
- 금액 단위는 만원 기준으로 입력한다

- 예상 최종 대출 가능액
- 디딤돌 예상 가능 여부
- 디딤돌 예상 가능액

- 결과 설명 문구

# 출력 우선순위
결과 화면 최상단에는 아래 순서로 보여준다.

2. 디딤돌 예상액
3. 보금자리론 예상액
4. 일반 주담대 예상액
5. 체증식 가능 여부
6. 보유 현금 포함 총 구매 가능 금액

# 1단계 계산 방향
1단계에서는 실제 금융기관 심사를 완전히 재현하지 않는다.
대신 아래 원칙으로 보수적인 예상치를 만든다.

- 소득 기준 상환 여력을 계산한다
- 주택 가격 기준 LTV 상한을 계산한다
- 상품별 최대 한도를 계산한다
- 최종 예상 대출액은 위 값 중 가장 보수적인 값을 사용한다

- 선택 가능한 대출 방식
을 바탕으로 가능 여부를 별도 표시한다.

# 결과 문구 예시
- 정부대출 가능성이 있습니다
- 생애최초 조건이 반영되면 한도 확대 가능성이 있습니다
- 주택 가격 기준으로 정부대출은 어려우나 일반 주담대는 검토 가능합니다
- 체증식은 조건부로 가능할 수 있습니다
- 자영업자는 소득 입증 상태에 따라 실제 결과가 달라질 수 있습니다

# 결과 화면에서 반드시 보여줄 안내
- 본 결과는 입력값 기준 예상치입니다.
- 실제 대출 가능 여부와 금리는 금융기관 심사 및 최신 규제에 따라 달라질 수 있습니다.
- 정책대출 자격은 실제 신청 시점의 기준으로 다시 확인이 필요합니다.

# 1단계 범위에서 제외
- 로그인
- 실거래가 API 연동
- 지도
- 매물 리스트
- PDF 리포트

- 커뮤니티
- 고급 투자 분석

# 성공 기준
- 사용자가 3분 안에 입력을 마칠 수 있다
- 결과를 보고 가능/불가능을 직관적으로 이해할 수 있다
- 부족 금액이 명확하게 보인다
- 조건을 바꾸면 결과가 달라진다
- 실제 상담 전에 헛발품을 줄이는 데 도움이 된다
