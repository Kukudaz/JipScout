# 서비스명
내 대출 가능액 판정기

# 한 줄 설명
대한민국 실거주 사용자가 소득, 무주택 여부, 생애최초 여부, 출산 여부, 직업 형태, 주택 가격 등을 입력하면
신생아 특례 디딤돌, 디딤돌, 보금자리론, 일반 주담대의 가능 여부와 예상 한도를 빠르게 보여주는 웹앱

# 문제 정의
기존 부동산 서비스는 매물 정보는 많지만,
사용자가 실제로 가장 먼저 궁금해하는 아래 질문에 즉답하지 못한다.
- 나는 대출이 되나
- 어떤 상품이 되나
- 왜 안 되나
- 얼마까지 되나
- 체증식이 되나

# 1단계 목표
첫 화면에서 핵심 정보만 받고 아래를 바로 보여준다.
- 예상 최종 대출 가능액
- 신생아 특례 디딤돌 가능 여부 / 예상액 / 불가 사유
- 디딤돌 가능 여부 / 예상액 / 불가 사유
- 보금자리론 가능 여부 / 예상액 / 불가 사유
- 일반 주담대 예상 가능액
- 체증식 가능 여부
- 보유 현금 포함 총 구매 가능 금액

# 핵심 사용자
- 생애최초 실수요자
- 신혼부부
- 출산가구
- 일반 직장인
- 자영업자 / 프리랜서
- 부동산 상담 전에 자기 가능성을 먼저 보고 싶은 사람

# 사용자 입력
## 기본 정보
- 본인 연소득
- 배우자 연소득
- 보유 현금
- 기존 월 부채 상환액
- 나이
- 직업 형태
- 혼인 상태
- 무주택 여부
- 생애최초 여부
- 자녀 수
- 대출신청일 기준 2년 내 출산 여부
- 체증식 관심 여부

## 주택 정보
- 희망 주택 가격
- 주택 전용면적
- 수도권 여부
- 규제지역 여부

# 입력 정책
- 금액은 만원 단위로 입력
- 숫자 입력 기본값 0 강제 금지
- 입력칸은 비어 있을 수 있어야 함
- 계산 시에만 숫자로 변환
- 비어 있는 숫자는 0으로 안전 변환

# 반드시 지원할 결과
## 1. 예상 최종 대출 가능액
여러 상품 중 현실적으로 가장 유력한 예상 금액

## 2. 상품별 결과 카드
각 카드에 아래 정보 포함
- 상품명
- 상태: 가능 / 조건부 / 어려움
- 예상 가능액
- 핵심 사유 1~3개
- 대표 불가 사유
- 추천 상환 방식 또는 체증식 가능 여부

## 3. 총 구매 가능 금액
- 보유 현금
- 예상 최종 대출 가능액
- 둘을 합친 총 구매 가능 금액

# 상품별 판정 엔진 요구사항
## 신생아 특례 디딤돌
- 출산 여부를 가장 먼저 본다
- 소득, 무주택 여부, 주택 가격, 면적, 지역 조건을 함께 본다
- 예상액과 불가 사유를 동시에 보여준다

## 디딤돌
- 무주택 여부, 생애최초 여부, 혼인 상태, 소득, 주택 가격을 본다
- 예상액과 불가 사유를 동시에 보여준다

## 보금자리론
- 주택 가격, 무주택/1주택 여부, 소득, 생애최초 여부를 본다
- 예상액과 불가 사유를 동시에 보여준다
- 만기 40년/50년 가능성은 나이와 혼인 상태를 참고 문구로 보여준다

## 일반 주담대
- 정책대출과 별개로 보수적 추정 엔진을 둔다
- 소득과 기존 월 부채 상환액 기반 상환 여력으로 예상액을 계산한다
- 규제지역/비규제지역에 따른 LTV 차이를 반영 가능한 구조로 만든다

## 체증식
- 별도 결과로 분리한다
- 상품별 허용 조건을 반영한다
- 가능 / 조건부 / 어려움으로 보여준다

# 결과 문구 예시
- 신생아 특례 디딤돌 가능성이 있습니다
- 디딤돌은 주택 가격 조건 때문에 어렵습니다
- 보금자리론은 가능하나 소득과 LTV를 함께 확인해야 합니다
- 일반 주담대는 보수적으로 이 정도 수준이 예상됩니다
- 체증식은 나이와 상품 조건 기준으로 가능합니다
- 자영업자는 실제 소득증빙 상태에 따라 차이가 날 수 있습니다

# 1단계 제외 항목
- 매물 추천
- 상세 지역 비교
- 평면/향/역세권 비교
- 지도
- 실거래가 API
- 인테리어 비용 계산
- 결혼비용 계산
- PDF 리포트
- 회원 기능
- 커뮤니티

# 성공 기준
- 입력이 1분 내 끝난다
- 결과 보고 왜 되는지/왜 안 되는지 이해된다
- 단순 가능/불가가 아니라 금액과 사유가 나온다
- 출산가구, 생애최초, 신혼, 자영업자 차이를 드러낸다
- 한국 사용자가 "이제 좀 현실적이다"라고 느낀다
